  Option 1: Local AI (FREE, Private)

    - Select "Ollama" in Settings
    - 100% free, runs on your computer!

  1. Click Settings (⚙️) from dashboard

  3. Choose your model

🆘 TROUBLESHOOTING

  Problem: Browser doesn't open
  Solution: Manually go to http://localhost:5173

  Problem: Can't connect to AI
  Solution: Check Settings, verify API key or Ollama is running

  Problem: "Gateway disconnected"
  Solution: Make sure nullclaw.exe is in the same folder
